React

練習部署 React 官網上的圈圈叉叉小遊戲到 GitHub 上並參照指南後面要求增加新的功能 :

新增功能 :

  1. 在歷史動作列表中，用（欄，列）的格式來顯示每個動作的位置。
  2. 在動作列表中，將目前被選取的項目加粗。
  3. 加上一個切換按鈕讓你可以根據每個動作由小到大、由大到小來排序。
  4. 當勝負揭曉時，把連成一條線的那三個方格凸顯出來。
  5. 當沒有勝負時，顯示遊戲結果為平手。

程式碼連結：https://github.com/chenalex207/react-game
